# Inoffizielle-Scrappbook-Tabellenansicht
Ein inoffizielles Browser-Add-on für Scrappbook, das Kommentare und Favoriten aus einer Auswahl in einer übersichtlichen Tabellenansicht darstellt. Damit können alle Bilder, Kommentare und Likes schnell verglichen, sortiert und exportiert werden (z. B. als CSV oder Markdown-Datei).
